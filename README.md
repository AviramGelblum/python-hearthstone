## python-hearthstone

A Hearthstone Python library, including a CardDefs.xml parser
and Hearthstone enums.


### Requirements

* Python 3.4+

### Installation

* To install from source: `./bootstrap && ./setup.py install`
* To install from pip: `pip install hearthstone`
